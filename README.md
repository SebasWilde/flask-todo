# Flask TODO app

---
## Install the project

### 1 Clone the repo


### 2 Create virtual env

    virtualenv venv --python=python3.8

### 3 Run the following command
    $ pip install -r requirements.txt

## Run the project

    $ export FLASK_APP=main
    $ export FLASK_DEBUG=1
    $ export FLASK_ENV=development
    $ flask run

## Run test
    $ export FLASK_APP=main
    $ flask test
